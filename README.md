# Лабиринт сокровищ
Текстовая приключенческая игра на Python, где игрок исследует лабиринт, решает головоломки и ищет сокровища.
## Описание
"Лабиринт сокровищ" — это консольная игра-квест, в которой вы путешествуете по комнатам таинственного лабиринта. Вам предстоит:
- Исследовать различные локации
- Собирать предметы и управлять инвентарем
- Решать головоломки и загадки
- Избегать ловушек
- Найти путь к сокровищнице и открыть заветный сундук
## Особенности
- **8 уникальных комнат**: от входа в лабиринт до секретной сокровищницы
- **Система головоломок**: решайте загадки, чтобы получить награды
- **Инвентарь**: собирайте и используйте предметы
- **Случайные события**: каждое прохождение уникально
- **Ловушки**: будьте осторожны в опасных комнатах
- **Альтернативные решения**: несколько путей к победе
## Установка
### Требования
- Python 3.12+
- Poetry
### Установка зависимостей
```bash
make install

Запуск игры
make project

Управление
Доступные команды:

go <direction> или просто north/south/east/west — перейти в направлении
look — осмотреть текущую комнату
take <item> — поднять предмет
use <item> — использовать предмет из инвентаря
inventory — показать содержимое инвентаря
solve — попытаться решить загадку в комнате
help — показать справку по командам
quit — выйти из игры
Советы для прохождения
Внимательно читайте описания комнат
Собирайте все предметы — они могут пригодиться
Решайте головоломки, чтобы получить награды
Будьте осторожны в комнате с ловушками
Для входа в сокровищницу нужен специальный ключ
